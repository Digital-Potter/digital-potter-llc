CMS modeling for these plans

Each plan card is one entry in theDavid SubscriptionPlan module — title, slug, features list, "starts at" price (display-only), and a category tag (Restaurant / Ecommerce / Booking / Subscription / All / Enterprise). The Pricing page fetches the list at build time. You can edit them from the CMS without redeploying the frontend.
